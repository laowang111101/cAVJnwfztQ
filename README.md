# 项目简介
考研图书推荐系统，包涵算法，毕业项目，计算机的，springboot+vue前后端分离。本项目基于Spring Boot、Vue渐进式框架以及协同过滤算法，开发并实现一款可靠的考研辅导推荐系统。使用Python网络爬虫当当网获取教辅信息并存储到MySQL数据库，利用Redis缓存提高系统性能，用MD5工具对文件进行存储优化，利用Token进行用户身份验证。系统的设计架构涵盖了前端界面的呈现与用户互动以及后端数据处理与推荐算法的实现。在前台，构建基于用户推荐和物品推荐Mahout框架的协同过滤算法引擎，导入用户兴趣和行为历史数据构建用户物品矩阵，计算相似度构建相似矩阵，生成推荐结果再进行模型评估，提供给用户推荐的教辅、热榜教辅等，从而为用户提供个性化的教辅推荐。通过实验和评估，系统呈现出良好的推荐效果和用户体验，有效支持和指导考研学习。


## 核心功能
- **协同过滤算法引擎**：基于用户推荐和物品推荐，通过Mahout框架实现。
- **用户行为数据采集**：收集用户兴趣和行为历史数据，构建用户物品矩阵。
- **相似度计算**：计算用户与物品之间的相似度，构建相似矩阵。
- **推荐结果生成**：根据相似度矩阵生成个性化的教辅推荐结果。

## 适用场景
- **考研复习指导**：为考研学子提供个性化的教辅推荐，辅助复习。
- **教辅信息查询**：用户可查询相关教辅信息，了解书籍详情。
- **用户互动交流**：用户可参与互动，分享学习心得。

## 技术特点
- **Spring Boot**：作为后端框架，提供稳定的服务器端支持。
- **Vue**：使用Vue进行前端开发，实现前后端分离。
- **MySQL数据库**：存储教辅信息及相关数据。
- **Redis缓存**：提高系统性能，优化数据读取速度。
- **MD5加密**：对文件进行存储优化，保证数据安全。
- **Token验证**：实现用户身份验证，确保系统安全。
